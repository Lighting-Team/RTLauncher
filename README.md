## 简介 (Introduction)

**RTLauncher** 是一个基于现代化技术栈构建的 Minecraft 启动器。它结合了 **Next.js** 的灵活性与 **Rust** (通过 **Tauri**) 的高性能，旨在为玩家提供流畅、美观且资源占用极低的游戏启动体验。

本项目致力于打造一个易于扩展、界面优雅且功能强大的启动器核心。

## 核心特性 (Features)

* 🚀 **高性能**: 基于 Rust 编写核心逻辑，资源占用极低。
* 🎨 **现代化界面**: 使用 Next.js + Shadcn UI 构建的现代化、响应式用户界面。
* 🛠️ **深度集成**: 前后端通过 Tauri 高效通信，提供原生般的体验。
* ⚙️ **配置管理**: 完善的配置文件管理和初始化流程。
